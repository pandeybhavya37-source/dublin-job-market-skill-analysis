# Dublin Job Market Skill Analysis

## Overview
This project analyzes job postings in Dublin to identify the most in-demand technical skills in the technology industry.

## Goal
Understand which programming languages and tools appear most frequently in job descriptions.

## Technologies Used
- Python
- Pandas
- BeautifulSoup
- Matplotlib
- Seaborn

## Method
1. Collected job postings from online job boards
2. Extracted required technical skills
3. Counted frequency of each skill
4. Visualized results using Python libraries

## Results
The analysis shows that Python, SQL, and cloud technologies appear frequently in job listings.
